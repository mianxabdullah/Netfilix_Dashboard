# 🎬 Netflix Content Analytics Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of Netflix content using interactive visualizations and business insights. The dashboard explores content distribution, genre trends, ratings, countries, directors, and cast information to uncover patterns within Netflix's content library.

The project was built using Power BI, Power Query, and DAX measures, with a custom Netflix-inspired theme for a professional and engaging user experience.

---

## 🎯 Objectives

- Analyze Netflix content distribution across Movies and TV Shows.
- Identify popular genres, ratings, and producing countries.
- Explore content growth trends over time.
- Analyze director and cast contributions.
- Provide interactive insights through filters and navigation.

---

## 🔧 Data Preparation

The dataset was cleaned and transformed using Power Query:

- Removed null values
- Standardized genres
- Created Primary Genre column
- Processed cast and director information
- Generated calculated columns
- Built DAX measures for KPIs

---

## 📈 Key Insights

- Movies dominate Netflix's content library.
- Drama is among the most popular genres.
- TV-MA is one of the most common content ratings.
- Netflix content growth accelerated significantly after 2015.
- The United States contributes the largest share of content.
- Certain directors and actors appear frequently across titles.
